# BIA.ge Scraper


პროექტის გასაშვებად ბრძანება:

```bash
npm run scrape
```

## კომპანიების რაოდენობის შეზღუდვა

კომპანიების მაქსიმალური რაოდენობა კონფიგურირებულია `src/config/config.ts` ფაილში `maxCompanies` პარამეტრით. 

მაგალითად, თუ გვინდა მხოლოდ 100 კომპანიის მოგროვება:

```typescript
maxCompanies: 100
```

ამჟამად დაყენებულია 6000 კომპანია.
